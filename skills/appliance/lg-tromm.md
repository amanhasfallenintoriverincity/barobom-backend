# LG 트롬 세탁기 사용하기

## ⚠️ 먼저 확인하세요
- 세탁기 문이 완전히 닫혔는지 확인하세요
- 세제 투입구에 세제를 넣었는지 확인하세요
- 세탁 중에는 문이 잠기므로 강제로 열지 마세요

## 화면 구성
1. **좌측**: 원형 다이얼 (코스 선택)
2. **중앙**: 디스플레이 (남은 시간, 선택된 코스)
3. **우측 하단**: 시작/일시정지 버튼
4. **우측 상단**: 전원 버튼

## 자주 하는 일

### 표준 세탁하기
1. 전원 버튼을 누르세요
2. 다이얼을 돌려 "표준세탁"을 선택하세요
3. 디스플레이에 예상 시간이 표시됩니다
4. 시작/일시정지 버튼을 누르세요
5. 완료 알림이 울릴 때까지 기다리세요

### 탈수만 하기
1. 전원 버튼을 누르세요
2. 다이얼을 돌려 "탈수"를 선택하세요
3. 시작/일시정지 버튼을 누르세요

### 예약 세탁하기
1. 전원 버튼을 누르세요
2. 다이얼로 원하는 세탁 코스를 선택하세요
3. "예약" 버튼을 눌러 희망 완료 시간을 설정하세요
4. 시작 버튼을 누르면 설정한 시간에 맞춰 완료됩니다

## 문제 해결
- **문이 안 열릴 때**: 세탁이 완전히 끝난 후 1-2분 기다리면 열립니다
- **에러 코드가 뜰 때**: 표시된 번호를 메모하고 LG 서비스센터(1544-7777)에 문의하세요
