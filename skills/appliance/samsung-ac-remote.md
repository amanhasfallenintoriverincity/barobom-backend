# 삼성 무풍 에어컨 리모컨 사용하기

## ⚠️ 먼저 확인하세요
- 리모컨 건전지가 있는지 확인하세요
- 에어컨 본체에 리모컨을 향하게 하고 누르세요
- 실외기가 돌아가는 소리가 나면 정상 작동 중입니다

## 화면 구성
1. **상단 LCD**: 현재 온도, 설정 온도, 운전 모드
2. **좌측**: 전원, 온도 조절
3. **중앙**: 바람 세기, 바람 방향
4. **우측**: 모드 선택, 무풍, 예약

## 자주 하는 일

### 냉방 켜기
1. "전원" 버튼을 누르세요
2. "모드" 버튼을 눌러 "냉방"을 선택하세요 (눈송이 아이콘)
3. 온도 ↓ 버튼으로 원하는 온도를 설정하세요 (26도 권장)
4. 바람 세기로 원하는 세기를 고르세요

### 온도 올리기/내리기
- **온도 ↑**: 설정 온도가 1도 올라갑니다
- **온도 ↓**: 설정 온도가 1도 내려갑니다
- LCD 화면의 숫자가 변경되는지 확인하세요

### 무풍 모드 켜기
1. 냉방이 켜진 상태에서 "무풍" 버튼을 누르세요
2. 찬 바람이 직접 나오지 않고 간접 냉방됩니다
3. 다시 누르면 일반 냉방으로 돌아옵니다

### 끄기
- 전원 버튼을 한 번 누르면 꺼집니다
- 바로 꺼지지 않고 실외기가 천천히 멈추니 기다리세요

## 문제 해결
- **시원하지 않을 때**: 설정 온도를 확인하세요. 26도 이하로 설정되어 있는지, 냉방 모드인지 확인
- **리모컨이 안 될 때**: 건전지를 교체해보세요 (AAA 2개)
- **물이 떨어질 때**: 실내기에서 물방울이 떨어지면 필터 청소가 필요합니다
