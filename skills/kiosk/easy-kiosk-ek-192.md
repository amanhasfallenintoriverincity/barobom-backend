# 메가MGC커피 키오스크로 주문하기 (EK-192)

## ⚠️ 먼저 확인하세요
- 결제 전에 장바구니 내용을 꼭 확인하세요
- 애플페이는 왼쪽 별도 단말기로 결제합니다
- 영수증은 자동으로 출력됩니다

## 화면 구성
1. **상단**: 메가MGC커피 로고, 매장 정보
2. **중앙**: 메뉴 카테고리 (커피, 음료, 디저트 등)
3. **하단 좌측**: 장바구니 + 주문 내역
4. **하단 우측**: 결제 버튼
5. **좌측 하단 별도**: 애플페이 결제 단말기

## 자주 하는 일

### 커피 주문하기
1. 중앙 화면에서 원하는 메뉴 카테고리를 누르세요
2. 메뉴 목록에서 원하는 음료를 선택하세요
3. 사이즈와 옵션(샷 추가, 시럽 등)을 선택하세요
4. "담기" 버튼을 누르세요
5. 다른 메뉴도 같은 방법으로 추가하세요
6. 하단 "주문하기" 버튼을 누르세요

### 결제하기
- **신용카드**: 화면 안내에 따라 카드를 꽂거나 찍으세요
- **애플페이**: 왼쪽에 있는 별도 단말기에 아이폰을 갖다 대세요
- 결제 완료 후 영수증이 자동 출력됩니다

### 포인트 적립하기
- 결제 전 화면에서 "포인트 적립" 버튼을 누르세요
- 전화번호를 입력하거나 포인트 카드를 찍으세요

## 문제 해결
- **화면이 멈췄을 때**: 직원을 불러주세요. 혼자 해결하려고 오래 기다리지 마세요
- **잘못 담았을 때**: 장바구니에서 해당 메뉴 옆 "삭제" 버튼을 누르세요
- **결제가 안 될 때**: 카드 방향을 바꿔보거나 직원에게 도움을 요청하세요
