# 귀뚜라미 보일러 온도조절기 사용하기

## ⚠️ 먼저 확인하세요
- 보일러 본체 전원이 켜져 있는지 확인하세요
- 가스 밸브가 열려 있는지 확인하세요
- 겨울철에는 동파 방지를 위해 전원을 끄지 마세요

## 화면 구성
1. **상단**: 현재 실내 온도
2. **중앙**: LCD 디스플레이 (설정 온도, 모드)
3. **하단 버튼**: 난방, 온수, 외출, 예약, 온도 ↑↓

## 자주 하는 일

### 난방 켜기
1. "난방" 버튼을 누르세요
2. 디스플레이에 현재 설정 온도가 표시됩니다
3. 온도 ↑↓ 버튼으로 원하는 온도를 설정하세요 (20-22도 권장)
4. 보일러가 가동되면 따뜻한 바람이 나옵니다

### 온도 올리기/내리기
- **온도 ↑**: 설정 온도가 올라갑니다 (최대 40도)
- **온도 ↓**: 설정 온도가 내려갑니다 (최소 10도)
- 실내 온도가 설정 온도에 도달하면 보일러가 자동으로 멈춥니다

### 온수 켜기
1. "온수" 버튼을 누르세요
2. 온수 온도가 표시됩니다
3. 필요시 온도 조절이 가능합니다
4. 수도꼭지를 온수 방향으로 틀면 따뜻한 물이 나옵니다

### 외출 모드
1. "외출" 버튼을 누르면 외출 모드가 켜집니다
2. 실내 온도를 8-10도로 유지해 동파를 방지합니다
3. 다시 "난방"을 누르면 일반 모드로 돌아옵니다

## 문제 해결
- **난방이 안 될 때**: 설정 온도가 현재 실내 온도보다 높은지 확인하세요
- **온수가 안 나올 때**: "온수" 버튼이 켜져 있는지 확인하세요
- **에러 코드**: 디스플레이에 숫자나 글자가 깜빡이면 귀뚜라미 AS(1588-9000)로 연락하세요
